# Hospital Management System

## 📝 About the Project
The Hospital Management System is a beginner-friendly C application that allows users to manage patient records efficiently. The application provides a menu-driven interface to add, display, search, update, and delete patient information.

## ✨ Features
- Add patient records
- Display all patients
- Search patient by ID
- Update patient details
- Delete patient records
- Simple menu-driven interface

## 🛠️ Technologies Used
- C Programming
- Structures
- Arrays
- Functions
- Menu-Driven Programming

## 🎯 Learning Outcomes
This project helped me learn:
- Working with structures in C
- Implementing CRUD (Create, Read, Update, Delete) operations
- Using functions for modular programming
- Searching and updating records
- Developing menu-driven console applications

## 🚀 Future Improvements
- Store patient records using file handling
- Add doctor management
- Implement appointment scheduling
- Generate patient reports
- Improve input validation

## 📖 About
A simple C-based Hospital Management System for managing patient records using structures and menu-driven programming.
